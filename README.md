## ドキュメント

### SortedSet

ソート済み列をいくつかのバケットに分割して管理します。このとき、(バケットの個数) : (バケット内の個数) = 1 : 50 くらいにします。(insert / erase の定数倍が軽く、バケット再構築の定数倍が重いため)
あるバケットが空になったり、多すぎたりしたら、1 度まとめて、均等にバケットに分割します。
基本的に、要素の変更を伴う操作は O(√N) 、伴わない操作は O(log N) と思って良いです。
